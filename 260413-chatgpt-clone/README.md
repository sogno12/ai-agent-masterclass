# 💡 AI 라이프 코치 (ChatGPT Clone)

본 프로젝트는 로컬 LLM(Ollama)과 웹 검색 도구를 조합하여 구성된 **나만의 개인 맞춤형 라이프 코치 AI** 애플리케이션입니다. 

## 📆 개발 기록
- **최종 업데이트 일자:** 2026년 4월 19일

## ✨ 구현된 핵심 기능

### 1. 강력한 로컬 AI와 웹 검색 (Agentic Architecture)
- `gemma4:e4b` 기반의 로컬 고성능 언어모델 사용 (`openai-agents` SDK 연동)
- **도구 호출(Tool Calling):** 단순한 조언을 넘어, 최신 트렌드나 명언, 정확한 정보가 필요할 경우 스스로 **웹 검색 도구(`duckduckgo_search`)**를 호출하여 팩트 기반의 답변을 제공합니다.
- 콘솔 및 UI 상에서 모델이 어떤 키워드로 웹 검색을 했는지 시각적으로 확인할 수 있습니다.

### 2. SQLite 기반 다중 세션 메모리 (Persistent Sessions)
- **과거 대화 보존:** 앱을 재시작해도 모든 대화가 `sessions.db` 파일에 영구적으로 안전하게 보관됩니다.
- **다중 대화방 관리:** 사이드바(Sidebar)를 통해 여러 개의 주제별 대화방을 파고, 언제든 예전 대화로 자유롭게 이동할 수 있습니다.
- **지연 생성(Lazy Creation):** 쓸모없는 깡통 대화방이 무한 증식하는 것을 막기 위해, '새 대화' 탭을 활성화한 뒤 사용자가 첫 마디를 완전히 입력하는 순간에만 실제 DB 세션이 발급됩니다.

### 3. 세밀한 UX 개선 및 디버깅 툴
- **UI 잔상(Ghosting) 완전 제거:** Streamlit 특유의 무거운 동기식 렌더링 찌꺼기를 백그라운드 트릭으로 밀어내어 화면 겹침 현상을 완벽히 치유했습니다.
- **메모리 강제 초기화:** 한 번의 클릭을 통해 전체 데이터베이스(`DROP TABLE`) 및 UI 상태를 빈 화면으로 깔끔하게 포맷할 수 있는 기능을 제공합니다.
- **메모리 디버깅 뷰어:** 현재 진행 중인 대화 방 내에서 LLM이 실제로 유지하고 있는 과거 이력(JSON)을 사이드바 하단에서 들여다볼 수 있습니다.

### 4. VectorDB 기반 개인 기록(RAG) 검색
- **문서 업로드 및 기억:** 사이드바를 통해 텍스트로 된 일기장이나 목표(`*.txt`, `*.md`)를 업로드하면 무료 로컬 Vector DB(`ChromaDB`)를 통해 영구 저장됩니다.
- **100% 로컬 무료 구동:** 외부 OpenAI API 등을 사용하지 않고, 오직 Ollama의 한국어 특화 임베딩 모델(`bona/bge-m3-korean` 등)에 의존해 텍스트 문맥을 분석합니다.
- **AI 스마트 기록 연동:** 사용자 질문이 과거 이력이나 조언과 관련 있다고 판단될 경우 에이전트가 스스로 Vector DB를 파헤쳐 개인 문서를 찾아 읽고 맥락 맞춤형 코칭을 제공합니다.

### 5. 로컬 이미지 생성 및 비전 보드 시각화 (Local Image Generation)
- **100% 무료 로컬 이미지 백엔드:** **Stability Matrix**를 통해 격리된 환경에서 **Stable Diffusion WebUI(Automatic1111)**를 구동하며, 별도의 유료 API 결제 없이 무제한으로 이미지를 생성합니다.
- **최신 GPU(RTX 5060) 최적화:** 최신 아키텍처 지원을 위해 호환성이 떨어지는 `xformers` 대신 PyTorch 내장 **SDP Attention(`--opt-sdp-attention`)** 기술을 적용했습니다. 또한, `--medvram` 옵션을 통해 8GB VRAM 환경에서도 Ollama와 Stable Diffusion이 메모리를 나누어 쓰며 충돌 없이 안정적으로 구동됩니다.
- **비동기 에이전트 워크플로우:** `httpx` 라이브러리를 활용한 **비동기(Async) 도구 호출**을 구현하여, 고해상도 이미지 생성 중에도 에이전트의 스레드가 멈추거나 화면이 프리징되는 데드락(Deadlock) 현상을 완벽히 방지했습니다.
- **스마트 프롬프트 엔지니어링:** 사용자가 한국어로 요청하면 에이전트가 내부적으로 시각화에 최적화된 **영어 키워드**로 번역하고, '귀여운 일러스트 스티커 스타일'의 프롬프트를 자동 주입하여 기괴함을 제거한 고품질 결과물을 도출합니다.
- **UI 자동 렌더링 시스템:** 에이전트가 로컬 `output/` 폴더에 저장한 이미지 경로를 응답하면, `app.py`에서 **정규표현식(Regex)**으로 경로를 실시간 탐지하여 채팅창 내에 즉시 `st.image`로 시각화합니다.

## 🛠️ 실행 방법

이 프로젝트는 빠르고 신뢰성 높은 환경 구성을 위해 `uv` 패키지 매니저를 사용하고 있습니다. 

1. 필요한 패키지 설치: `uv sync` (또는 `uv pip install -r requirements.txt`)
2. 애플리케이션 실행:
```bash
uv run streamlit run app.py
```
*(참고: Ollama가 켜져 있어야 하며 `OLLAMA_MODEL` 변수는 `.env`에서 관리됩니다.)*
